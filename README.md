# Pharmaceutical AI

In this repository, I reproduce AI/machine learing papers for drug discovery.

## Reproductions

Currently available reproductions for:

- [Neural Message Passing for Quantum Chemistry](https://proceedings.mlr.press/v70/gilmer17a.html), ICML 2017 · [arXiv](https://arxiv.org/abs/1704.01212)
- **GraphVAE: Towards Generation of Small Graphs Using Variational Autoencoders**, 2018 · [arXiv](https://arxiv.org/abs/1802.03480) · [Authors' github](https://github.com/martin-simonovsky/graphvae)
- **Junction Tree Variational Autoencoder for Molecular Graph Generation**, ICML 2018 · [arXiv](https://arxiv.org/abs/1802.04364) · [Authors' github](https://github.com/wengong-jin/icml18-jtnn)
- **DeepAffinity: interpretable deep learning of compound–protein affinity through unified recurrent and convolutional neural networks** (*Bioinformatics*, 35(18), 3329–3338) · [Paper (PubMed Central)](https://pmc.ncbi.nlm.nih.gov/articles/PMC6755266/) · [Paper DOI](https://doi.org/10.1093/bioinformatics/btz111) · [Authors' github](https://github.com/Shen-Lab/DeepAffinity)
- **DeepDDS: deep graph neural network with attention mechanism to predict synergistic drug combinations** · [Paper DOI](https://doi.org/10.1093/bib/bbab390) · [Authors' github](https://github.com/Sinwang404/DeepDDs)
- **MolGAN: An implicit generative model for small molecular graphs** · [arXiv](https://arxiv.org/abs/1805.11973) · [Authors' github](https://github.com/nicola-decao/MolGAN)
